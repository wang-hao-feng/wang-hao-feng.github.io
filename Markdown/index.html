表格

语法:

code Markdown
表头1|表头2|表头3
---|:-:|--:
-一个|以上|表示分割表头和内容
文字|默认|居左
-右边|加:|表示右对齐
-两边|加:|表示居中

效果:

表头1表头2表头3
-一个以上表示分割表头和内容
文字默认居左
-右边加:表示右对齐
-两边加:表示居中
